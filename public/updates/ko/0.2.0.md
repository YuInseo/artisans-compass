# 업데이트 알림 (v0.2.0)
- **날짜**: 2026-02-10
- **내용**:

## 1. 윈도우 위치 버그 수정 (Window Positioning)
- **문제**: 위젯 모드에서 일반 모드로 돌아올 때, 창이 엉뚱한 모니터로 이동하는 현상 발생.
- **해결**: `electron/main.ts`에서 `screen.getDisplayMatching()`을 사용하여 현재 위젯이 있는 디스플레이를 정확히 감지한 후, 해당 화면의 중앙으로 복귀하도록 로직을 개선했습니다.

## 2. 테마 설정 전면 개편 (Theme Refactoring)
- **목표**: 메인 창과 위젯의 스타일(테마 + 컬러)을 완전히 분리.
- **작업 내용**:
    - `types.ts`: 위젯 전용 컬러 프리셋(`widgetThemePreset`) 저장 필드 추가.
    - `SettingsModal.tsx`: "모양(Appearance)" 탭을 **메인 창**과 **위젯** 섹션으로 명확히 분리.
    - `DailyPanel.tsx`: 위젯 모드 진입 시, 설정된 위젯 전용 컬러(CSS 변수)를 강제로 주입하여 메인 테마와 완전히 독립적인 스타일링이 가능하도록 구현.

## 3. 설정 사이드바 정리 (UI Polish)
- **요청**: "사용자 지정 명언" 카테고리 삭제 및 에디터 관련 설정 통합.
- **작업 내용**:
    - 설정 사이드바에서 "사용자 지정 명언" 메뉴 삭제.
    - 기존 "들여쓰기 가이드" 메뉴를 **"에디터 설정(Editor Settings)"**으로 이름 변경.
    - 해당 섹션에 "들여쓰기 가이드"와 "맞춤법 검사" 옵션을 함께 배치.

## 4. 기타 유지보수 (Maintenance)
- **번역(i18n)**: 새로 추가된 기능(에디터 설정, 위젯 테마 등)에 대한 한국어(`ko.json`) 번역 키 추가.
- **타입 오류 수정**: `themes.ts`의 `Theme` 인터페이스에 `description` 속성을 추가하여 빌드 에러 및 린트 경고 해결.
