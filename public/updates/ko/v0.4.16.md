# 업데이트 알림 (v0.4.16)
- **날짜**: 2026-03-09
- **내용**:
  ✨ **주요 기능 (Features)**
  - Todo 에디터 내 이미지 확대/축소/이동(Pan/Zoom) 기능을 지원하는 모달 뷰어 추가
  - 이미지 블록의 동적 크기 조절(핸들 드래그) 및 캡션/ALT 텍스트 작성 기능 지원
  - 전체 이미지 접기/펼치기 기능 추가
  - 로컬 디렉터리(`UserData/plugins`) 기반의 동적 플러그인 로드 및 설정 기능 추가
  - 선언적 JSX 기반 컴포넌트 플러그인 개발 API(`useArtisansCompass`) 도입 및 샌드박스 환경 구축

  🎨 **UI/UX 개선 (UI/UX Improvements)**
  - 노션(Notion) 스타일의 이미지 호버 툴바와 인접 레이아웃 적용 및 패딩/마진 리팩토링
  - 설정 모달의 플러그인 탭 다국어 번역 지원 및 레이아웃 정리
  - 앱 타이틀 바 및 캘린더 뷰 상단 헤더 정렬(Alignment) 미세 조정

  🔧 **시스템 및 성능 최적화 (System & Performance)**
  - 이미지 드래그/패닝 시 GPU 가속(`translate3d`)을 통한 60fps 부드러운 렌더링 최적화
  - `ArtisansCompassProvider` 및 플러그인 의존성 주입 아키텍처 전면 개편
  - 플러그인 Context 분리 및 캡슐화(`IArtisansEngine`)로 보안성과 안정성 강화

  🐛 **버그 수정 (Bug Fixes)**
  - `local-image://` 커스텀 프로토콜을 도입하여 에디터 로컬 이미지 로딩 버그 수정
  - 이미지 드래그 시 텍스트 선택(Highlighting) 버그 및 드래그 멈춤 현상 해결
  - JSX 플러그인 환경에서의 React 상태 업데이트 충돌 및 무한 렌더링(JS Heap Out of memory) 버그 해결
