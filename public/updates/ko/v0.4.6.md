# 업데이트 알림 (v0.4.6)
- **날짜**: 2026-02-28
- **내용**:

# 사이드바 캘린더 위젯 팝업 기능 추가

기존 데일리 퀘스트 뷰에 있던 캘린더 화면을 좌측 사이드바(메뉴 영역)로 분리하였습니다. 이제 캘린더 아이콘을 클릭하여 언제든지 일정과 아카이브 된 기록을 통계처럼 간편하게 조회할 수 있습니다. 데일리 퀘스트 작성 페이지 또한 더욱 깔끔하게 유지됩니다.


# 통계 및 아카이브 전용 뷰 추가

이전에 팝업 형태로 제공되던 캘린더 기능을 화면 전체를 사용하는 전용 뷰로 변경했습니다. 사이드바 메뉴의 새로운 통계 아이콘(BarChart)을 클릭하여, 더욱 넓고 직관적인 환경에서 과거 일정을 확인하고 기록된 데이터를 돌아볼 수 있습니다.


# 통계 뷰 레이아웃 개선

통계 및 아카이브 뷰의 헤더 디자인을 개선하여, 화면 우측 상단에서 날짜를 직관적으로 변경할 수 있도록 네비게이션 컨트롤을 이동시켰습니다. 이제 기존의 데일리 아카이브 모달과 동일하게 일관된 사용자 경험을 제공합니다.


# 통계 뷰 레이아웃 최종 개선

통계 및 아카이브 뷰의 상단 헤더 영역("Statistics & Archive" 텍스트 포함)을 완전히 제거하여 화면 공간 활용도를 높였습니다. 또한, 날짜 변경 네비게이션 컨트롤을 메인 상단 툴바 영역의 가장 우측으로 이동시켜, 앱의 전반적인 UI와 더 자연스럽게 어우러지도록 개선했습니다.


### 일일 퀘스트 통합
- 목표 보기 모드에서 일반 작업 목록의 제목이 현재 설정된 일일 퀘스트로 표시되도록 개선했습니다.
- 위젯 모드에서 프로젝트가 선택되지 않았을 때 기본 작업 목록 대신 일일 퀘스트 상태 창이 직관적으로 표시되도록 UI를 수정했습니다.


### 일일 퀘스트 및 일반 작업 뷰 개선
- 목표 보기 화면에서 일반 작업 목록 상단에 일일 퀘스트 설정 및 달성 현황 창(TodoSidebar)을 내장하여, 작업과 퀘스트를 한 눈에 볼 수 있도록 직관성을 높였습니다.
- 위젯 모드가 아닐 때 퀘스트 창이 위화감 없이 잘 어우러지도록 배경을 투명하게 처리하였습니다.


### 일일 퀘스트 및 일반 작업 뷰 레이아웃 개선
- 사용자의 피드백을 반영하여 목표 보기 화면에서 일반 작업 목록이 2단으로 나뉘던 것을, 기본 화면과 동일하게 하나의 작업 목록과 퀘스트 좌측 뷰가 유지되도록 통일하였습니다.
