# 업데이트 알림 (v0.3.3)
- **날짜**: 2026-02-20
- **내용**: 

### ✨ 새로운 기능
- **루틴 에디터 UI 개선**: 글래스모피즘(Glassmorphism) 디자인을 적용하여 더욱 현대적이고 세련된 UI로 개편했습니다.
- **기간 설정 탭 개선**: 시작/종료 시간을 직관적으로 설정할 수 있도록 입력 방식을 개선했습니다.
- **자동 저장 기능**: 루틴 편집 중 창을 닫거나 배경을 클릭하면 작성 중인 내용이 자동으로 저장됩니다.
- **스마트 팝오버 위치**: 화면 공간에 따라 팝오버가 왼쪽 또는 오른쪽에 최적의 위치로 표시됩니다.

### 🐛 버그 수정
- **드래그 앤 드롭 오류 수정**: 루틴 드래그 시 다른 요일의 일정까지 사라지거나, 원래 위치로 되돌아가는 현상을 수정했습니다.
- **모드 전환 시 선택 오류 수정**: 캘린더와 루틴 모드 전환 시 선택된 일정이 남아있는 문제를 해결했습니다.
