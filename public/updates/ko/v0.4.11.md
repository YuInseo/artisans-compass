# 업데이트 알림 (v0.4.11)
- **날짜**: 2026-03-03
- **내용**:



### 🍅 포모도로 컨텍스트 메뉴 & 설정 UI 개선

*   **포모도로 우클릭 옵션:** 포모도로 사이드바의 작업 항목을 우클릭하면 컨텍스트 메뉴가 열려 항목을 쉽게 삭제하거나 수정(예정)할 수 있도록 기능이 추가되었습니다. (`src/components/dashboard/PomodoroPanel.tsx`) 
*   **디스크 사용량 로딩 애니메이션:** 설정의 '스크린샷 저장 위치'에서 디스크 총량을 계산하는 동안 렉이 걸리지 않도록 로딩 스피너와 안내 문구가 추가되었습니다. (`src/components/settings/tracking-tab.tsx`)
*   **현지화 적용:** `ko`, `en`, `ja` 환경에 맞게 로딩 문구를 추가했습니다.




### 🍅 포모도로 편집 기능 추가

*   **편집 모달:** 포모도로 우클릭 컨텍스트 메뉴에서 `편집`을 클릭하면, 기존 작업을 수정할 수 있도록 `AddPomodoroTaskDialog`를 재사용하여 편집 모달이 열리도록 변경했습니다. (`src/components/dashboard/AddPomodoroTaskDialog.tsx`, `src/components/dashboard/PomodoroPanel.tsx`)
*   **다국어 지원:** 편집 모달의 제목 부분 다국어를 지원합니다. (`ko, en, ja`)




### ⚙️ 타임테이블 설정 UI 카테고리화

설정 화면의 타임테이블(Timeline) 설정 항목에 스위치들이 너무 많아 무엇을 설정하는지 한눈에 알아보기 어려웠던 점을 개선했습니다. 

*   **관련 설정 묶기:** 총 8개의 개별 스위치 설정들을 기능 성격에 맞게 3개의 카테고리로 묶어 디자인을 정돈했습니다.
    *   `타임테이블 뷰 및 표시항목`
    *   `시간 안내선 및 마커`
    *   `상호작용 및 동작 제어`
*   **다국어 지원 (ko, en, ja):** 각각의 새로운 카테고리 제목에 다국어 번역을 추가했습니다. (`src/components/settings/timeline-tab.tsx`, `locales/*.json`)




### 🐛 타이틀 바 및 윈도우 컨트롤 UI 개선

타이틀 바 우측의 버튼들(새 프로젝트, 창 닫기/최대화/최소화 버튼)의 높이가 부모 컨테이너에 맞춰지지 않아 찌그러져(Squished) 보이는 현상을 수정했습니다.

*   **AppTitleBarControls.tsx 수정:** 우측 컨트롤 패널의 루트 `div`에 `h-full` 속성을 부여하여 부모인 `AppTitleBar`(높이 `h-10`)의 전체 높이를 사용하도록 변경했습니다. 윈도우 컨트롤 버튼들에 적용된 `padding` 이슈를 해결하여 버튼 영역이 상하좌우를 온전히 채우도록 디자인을 개선했습니다.
