# アップデートのお知らせ (v0.2.4)
- **日付**: 2026-02-14
- **内容**:
  - **ダイナミックデイリーアーカイブモードの追加**:
    - 深夜まで作業する場合、アプリを終了するまで「同じ日」として記録を続ける設定が可能になりました。
    - 設定 > タイムテーブル > 「一日の記録モード」から**ダイナミック（アプリ終了まで）**を選択してください。
  - **タイムテーブル表示の改善**:
    - 24時間を超えるセッション（例：25:00、26:00）もグラフ上で自然に確認できるようになりました。
  - **設定UIの改善**:
    - 「一日の記録モード」の設定を**タイムテーブル**タブに移動し、アクセスしやすくしました。
  - **安定性の向上とバグ修正**。
