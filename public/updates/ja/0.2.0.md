# アップデートのお知らせ (v0.2.0)
- **日付**: 2026-02-10
- **内容**:

## 1. ウィンドウ位置のバグ修正 (Window Positioning)
- **問題**: ウィジェットモードから通常モードに戻る際、ウィンドウが別のモニターに移動してしまう現象が発生。
- **解決**: `electron/main.ts`で `screen.getDisplayMatching()` を使用して現在のウィジェットがあるディスプレイを正確に検出し、その画面の中央に戻るようにロジックを改善しました。

## 2. テーマ設定の全面改修 (Theme Refactoring)
- **目標**: メインウィンドウとウィジェットのスタイル（テーマ + カラー）を完全に分離。
- **作業内容**:
    - `types.ts`: ウィジェット専用のカラープリセット(`widgetThemePreset`)保存フィールドを追加。
    - `SettingsModal.tsx`: "外観(Appearance)"タブを **メインウィンドウ** と **ウィジェット** セクションに明確に分離。
    - `DailyPanel.tsx`: ウィジェットモードに入るとき、設定されたウィジェット専用カラー（CSS変数）を強制的に注入し、メインテーマと完全に独立したスタイリングが可能になるように実装。

## 3. 設定サイドバーの整理 (UI Polish)
- **要望**: "カスタム名言(Custom Quotes)"カテゴリーの削除およびエディタ関連設定の統合。
- **作業内容**:
    - 設定サイドバーから "カスタム名言" メニューを削除。
    - 既存の "インデントガイド" メニューを **"エディタ設定(Editor Settings)"** に名称変更。
    - そのセクションに "インデントガイド" と "スペルチェック" オプションをまとめて配置。

## 4. その他メンテナンス (Maintenance)
- **翻訳(i18n)**: 新しく追加された機能（エディタ設定、ウィジェットテーマなど）に対する韓国語(`ko.json`)翻訳キーを追加。
- **型エラー修正**: `themes.ts`の `Theme` インターフェースに `description` プロパティを追加し、ビルドエラーおよびLint警告を解決。
