# アップデートのお知らせ (v0.3.3)
- **日付**: 2026-02-20
- **内容**: 

### ✨ 新機能
- **ルーチンエディタUI刷新**: グラスモーフィズムデザインを適用し、よりモダンで洗練されたUIになりました。
- **期間設定タブの改善**: 開始・終了時間をより直感的に設定できるよう入力方式を改善しました。
- **自動保存**: エディタを閉じたり背景をクリックした際に、作業内容が自動的に保存されるようになりました。
- **スマートポップオーバー配置**: 画面のスペースに応じて、ポップオーバーが最適な位置（左または右）に表示されます。

### 🐛 バグ修正
- **ドラッグ＆ドロップの修正**: ルーチンをドラッグした際に他の曜日の予定が消えたり、元の位置に戻ったりする問題を修正しました。
- **モード切り替え時の選択解除**: カレンダーとルーチンモードを切り替える際に、選択状態が残ってしまう問題を解決しました。
