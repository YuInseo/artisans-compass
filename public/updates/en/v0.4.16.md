# Update Notice (v0.4.16)
- **Date**: 2026-03-09
- **Description**:
  ✨ **Features**
  - Added an in-app image viewer modal supporting pan, zoom, and dragging in the Todo Editor.
  - Supported dynamic image block resizing and adding caption/ALT texts.
  - Added a global feature to collapse and expand all image blocks at once.
  - Introduced dynamic local plugin loading from the `UserData/plugins` directory with settings management.
  - Launched a declarative JSX component-based API (`useArtisansCompass`) and sandbox environment for plugin developers.

  🎨 **UI/UX Improvements**
  - Revamped image block layouts and hover toolbars to match Notion's style for a cleaner look.
  - Improved layout alignments and removed padding/margin inconsistencies in headers and title bars.
  - Added multi-language translation support for the Settings modal's Plugin tab.

  🔧 **System & Performance**
  - Optimized image dragging and panning to silky smooth 60fps using GPU acceleration (`translate3d`).
  - Completely overhauled the `ArtisansCompassProvider` and core plugin dependency injection architecture.
  - Improved security and stability through interface encapsulation (`IArtisansEngine`) and isolated Context regions.

  🐛 **Bug Fixes**
  - Fixed issues with loading local images by introducing a dedicated `local-image://` custom protocol.
  - Fixed annoying text highlighting selections and lagging when dragging/resizing image blocks.
  - Resolved `JS Heap Out of memory` crashes and React infinite re-rendering loops related to JSX plugin rendering.
