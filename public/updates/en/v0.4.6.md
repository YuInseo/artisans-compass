# Update Notice (v0.4.6)
- **Date**: 2026-02-28
- **Description**:

# Add Calendar Widget Popover to Sidebar

The Calendar view, previously located within the Daily Quest view, has been moved to a separate popover on the left sidebar menu. You can now easily check your schedule and view archived history logs like statistics by simply clicking the calendar icon. This also keeps the Daily Quest page cleaner and more focused.


# Dedicated Statistics and Archive View

The calendar feature, previously available as a popup, has been upgraded to a dedicated full-screen view. By clicking the new statistics icon (BarChart) in the sidebar menu, you can now check your past schedules and review recorded data in a more spacious and intuitive environment.


# Statistics View Layout Refinement

Improved the header design of the Statistics and Archive view. The date navigation controls have been moved to the top right of the screen for more intuitive interaction. This provides a consistent user experience, mirroring the familiar layout of the Daily Archive modal.


# Final Statistics View Layout Refinement

Completely removed the top header area (including the "Statistics & Archive" text) from the Statistics and Archive view to maximize screen space utilization. Additionally, the date navigation controls have been relocated to the far right of the main top toolbar, integrating more naturally with the application's overall UI.


### Daily Quest Integration
- Improved the Goal View mode to correctly display the currently set daily quest as the title of the general tasks list.
- Updated the UI in Widget Mode to intuitively display the daily quest status window instead of the default task list when no project is selected.


### Daily Quest and General Task View Improvements
- Integrated the daily quest setting and status window (TodoSidebar) at the top of the general task list in the Goal View screen, improving intuitiveness by allowing users to see both tasks and quests at a glance.
- Ensured the quest window blends seamlessly without looking out of place by making its background transparent when not in widget mode.


### Daily Quest and General Task Layout Improvement
- Addressed user feedback by removing the split-column view in the Goal View. It now uniformly maintains a single task list alongside the left quest view, exactly matching the standard screen structure.
