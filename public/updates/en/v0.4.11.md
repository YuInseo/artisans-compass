# Update Notice (v0.4.11)
- **Date**: 2026-03-03
- **Description**:



### 🍅 Pomodoro Context Menu & Settings Loading UI

*   **Right-Click Menu:** Added a context menu to Pomodoro sidebar items, allowing users to easily delete or edit tasks. (`src/components/dashboard/PomodoroPanel.tsx`) 
*   **Disk Usage Loading State:** Added a loading spinner (`isCalculatingDiskUsage`) in the Screenshot settings tab to indicate when the folder size calculation is in progress. (`src/components/settings/tracking-tab.tsx`)
*   **Localization:** Added translation strings for the new loading state in `ko`, `en`, and `ja` locales.




### 🍅 Added Pomodoro Edit Functionality

*   **Edit Modal:** Clicking `Edit` from the Pomodoro context menu now opens the repurposed `AddPomodoroTaskDialog` pre-filled with the right configuration, allowing you to update the task setting directly. (`src/components/dashboard/AddPomodoroTaskDialog.tsx`, `src/components/dashboard/PomodoroPanel.tsx`)
*   **Localization Support:** Added edit string localization. (`ko, en, ja`)




### ⚙️ Categorized Timeline Settings UI

Improved the readability of the Timeline settings tab which previously had a long, unorganized list of toggle switches.

*   **Grouped Related Settings:** The 8 individual switch settings have been grouped into 3 logical categories to clean up the design:
    *   `View & Display`
    *   `Time Indicators & Markers`
    *   `Interaction & Restrictions`
*   **Localization (ko, en, ja):** Added localization support for the new category titles. (`src/components/settings/timeline-tab.tsx`, `locales/*.json`)




### 🐛 Title Bar and Window Controls UI Improvement

Fixed an issue where the buttons on the right side of the title bar (New Project, Close/Maximize/Minimize buttons) appeared squished because their heights were not aligning properly with the parent container.

*   **AppTitleBarControls.tsx Modifeid:** Added `h-full` to the root `div` of the right control panel so it properly inherits the `AppTitleBar`'s full height (`h-10`). Corrected the root padding so that Window controls touch the edges properly without unwanted compression.
