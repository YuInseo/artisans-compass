### ✨ 프로세스 이름 기반 앱 인식으로 변경
- **정확성 향상:** 이제 "현재 실행 중인 앱의 이름(Title)"이 아니라 "프로세스 이름(Process, 예: brave, Code 등)"만을 기준으로 앱을 식별하도록 백엔드 추적 로직을 완전히 교체했습니다.
- **불필요한 오작동 방지:** 웹 브라우저나 폴더 이름 등에 포함된 불필요한 키워드로 인해 잘못 기록되던 문제를 사전에 차단합니다. 설정의 '시간 추적 앱(Tracked Apps)'에 프로세스 이름(영어소문자 기준)을 등록해 편리하게 사용하세요.
- **타임테이블 표시 안내:** 현재 1분 13초(Antigravity 프로젝트)만 화면에 나왔던 이유는 설정의 **'작업 앱만 표시'** 옵션이 켜져 있었기 때문입니다! 시간은 전부 안전하게 기록되고 있으니, 'Brave' 등을 작업 프로그램에 등록하거나 해당 옵션을 끄면 모든 시간이 타임테이블에 정상 표시됩니다.
